# Installasi
## For Termux
```
> pkg update && pkg upgrade -y
> pkg install bash -y
> pkg install imagemagick -y
> pkg install ffmpeg -y
> pkg install nodejs -y
> pkg install git -y
> git clone https://github.com/talosmodders/LastMd
> cd LastMd
> npm install
> npm start
```
